# Template Blogar Style

Template Astro + Tailwind CSS para portal de notícias/revista moderna em dark mode.

## Recursos

- Astro com `output: "static"`.
- Tailwind CSS via `@tailwindcss/vite`.
- Content Collections em `src/content/blog/`.
- Home responsiva com hero de revista, cards com zoom, badges de categoria e newsletter.
- Página dinâmica de post em `src/pages/blog/[slug].astro`.

## Comandos

```bash
npm install
npm run dev
npm run build
```
